# @ocp-catalog/registration-schema

TypeScript and Zod schema package for `ocp.catalog.registration.v1`.

This package models the `Catalog Node -> OCP Catalog Registration` handshake:

- `RegistrationDiscovery`
- `RegistrationManifest`
- `CatalogRegistration`
- `CatalogRegistrationResult`
- `CatalogSearchRequest`
- `CatalogSearchResult`
- `CatalogRouteHint`
