## 申请步骤（5 步、按顺序做）

### 步骤 1 — 注册 / 登录京东联盟

打开 **https://union.jd.com**，用京东账号登录。
首次进入会要求实名认证（个人主体）或上传营业执照（公司主体）。

> ⚠️ 注意：用的京东账号会成为联盟账号主体，**请用一个长期可控的账号**（不要用临时账号或他人账号）。

### 步骤 2 — 填写联盟会员信息

主体审核通过后，进入"账户中心"→"会员信息"，填：
- 真实姓名 / 公司名（与实名一致）
- 收款方式（银行卡或支付宝；PoC 阶段填一个能收佣金的就行）
- 联系方式

完成这一步会拿到 **`UnionID`**（一串数字，10 位左右），后台首页会显示。**这是第 1 个要给技术的东西。**

### 步骤 3 — 创建媒体 + 应用（拿 AppKey/AppSecret）

进入"联盟 API"或"开发者中心"（不同时期叫法略有不同），按以下顺序：

1. **创建媒体**：选"网站"，填域名（如 `deeplumen.com`），描述写"AI Agent 推荐导购 PoC"
2. **创建应用**：关联到刚才的媒体，应用名 `ocp-jdunion-poc`，描述同上
3. 等待审核（**通常 1–2 工作日**）

审核通过后，应用详情页会有：
- **`AppKey`**（一串字母数字，约 16 位）
- **`AppSecret`**（一串字母数字，约 32 位）

**这是第 2、3 个要给技术的东西。**

### 步骤 4 — 创建推广位（拿 positionId）

进入"推广管理"→"推广位管理"→"添加推广位"：
- 推广位名称：`ocp-poc-001`
- 推广位类型：选"通用"或"商品推广"（任意能用的即可）
- 流量来源：选"网站"，关联到步骤 3 创建的媒体
- 描述：写"OCP Agent 推荐"

创建后会得到 **`positionId`**（一串数字，约 9–10 位）。**这是第 4 个要给技术的东西。**

> 💡 PoC 阶段只需要 **1 个 positionId** 就够。所有 Agent 共用这一个，订单先归到统一账户。
> 后期想做"千 Agent 千归因"需要批量建 positionId（一般要联盟流量达到一定门槛才开放批量申请）—— 但这不在 PoC 范围内。

### 步骤 5 — 准备 1 个测试 SKU

任选一个京东自营的稳定在售商品，从地址栏拿到 SKU ID。

形如：`https://item.jd.com/100012345678.html` → SKU ID = `100012345678`

**优先选择**：
- ✅ 自营商品（佣金率稳定、不易下架）
- ✅ 当前有佣金率（在京东联盟搜索"商品库"能看到佣金率的商品）
- ❌ 不要选预售 / 限时秒杀 / 第三方店铺

**建议**：选一个生活日用品（耳机 / 充电器 / 牙膏），便于演示时不引起隐私争议。

## FAQ

**Q：审核会被拒吗？常见原因是什么？**
A：审核相对宽松。常见拒审原因：
1. 媒体类型选了 APP / 小程序但没上架截图 → 改回"网站"重提
2. 应用描述太敷衍（如只写"测试"） → 写清楚"AI 导购 Agent 推荐链路 PoC"即可
3. 网站域名疑似空壳 → 填一个有备案的域名或公司站点

**Q：要花钱吗？**
A：**全程 0 元**。京东联盟是 CPS 分润模式（用户成交后从商家佣金里抽 10% 平台费），账号注册和 API 调用都免费。

**Q：调用 API 有频次限制吗？**
A：基础权限 `goods.query` 单 AppKey 默认 **1 万次/天**，足够 PoC 演示。
高频场景（>10 万次/天）需要申请提升，需要历史调用数据，PoC 阶段用不上。

**Q：佣金多久能到账？**
A：用户下单 → 7 天确认收货 → 月结（次月 20 号左右）→ 到账。
PoC 演示阶段拿不到结算佣金（演示时间太短），但能看到"预估佣金"实时累计。

**Q：测试时下单后能取消订单吗？要不要真花钱？**
A：演示时可以"加购物车 → 不下单"或"下单 → 立刻取消"，**不需要真实付款**。
联盟系统在用户下单瞬间就会记录归因，只要 PID 透传成功就能看到订单写入 ledger。

**Q：subUnionId 是什么？要不要顺便申请？**
A：JD 联盟支持 "subUnionId"（子渠道标识），用来做"千 Agent 千归因"——同一个 positionId 下，不同 Agent 用不同的 subUnionId，佣金能在账本里按 Agent 拆分。
**PoC 不需要额外申请**，subUnionId 是 API 调用时随便传的字符串，账号本身就支持。

**Q：拿到 AppKey 后多久能看到效果？**
A：技术拿到凭据后，**当天就能跑通真实链路**：
1. 把 5 个环境变量填进 `.env`
2. `JDUNION_MOCK=false`
3. `bun run smoke:jdunion`（mock smoke 改 1 行就变成真实 smoke）
4. 30 秒后能看到真 `u.jd.com` 短链，点击跳手 J 京东 App
